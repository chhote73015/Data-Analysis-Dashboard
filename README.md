# Gupta-Store-Data-Analysis(Interactive Dashboard creation using MS EXCEL)
## Project Objective 
The Gupta Store wants to create an annual sales report for 2022. So that owner of thr Gupta Store can understand their customer and grow more sales in 2023 & 2024.

## Dataset used
- <a href="https://github.com/rishabhnmishra/Excel_Vrinda_Store_Analysis/blob/main/Vrinda%20Store%20Data%20Analysis.xlsx">Dataset</a>


## Questions (KPIs)
- Compare the sales and orders using single chart.
- Which month get the higher sales and orders?
- Who purchase more - Men or Women?
- What are the different order status in 2022?
- list top 5 states contribution to the sales?
- Relation between age and gender based on number of orders.
- Which Channel is contributing maximum to the sales?
- Highest Selling category?
- Percentage of Total Orders Deliverd


- Dashboard Interaction  <a href="https://github.com/ritikbh193/Vrinda-Store-Data-Analysis/blob/main/Dashboard_Image.png">Dataset</a>


## Process
- Verify data for any missing values and anomalies, and sort out the same.
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created pivot tables according to the questions asked.
- Merge all pivot tables into one dashboard and apply slicer to make dynamic.
